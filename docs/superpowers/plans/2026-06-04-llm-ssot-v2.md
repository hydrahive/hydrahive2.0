# LLM-SSOT v2 — Implementierungsplan (Sub-Projekt 2: alle übrigen Picker auf die eine Quelle)

> **For agentic workers:** REQUIRED SUB-SKILL: superpowers:subagent-driven-development (empfohlen) oder superpowers:executing-plans, Task für Task. Steps mit Checkbox (`- [ ]`).

**Goal:** Jeder verbliebene Modell-Picker liest aus `GET /api/llm/models` (eine Quelle, sortiert); der alte `/llm`+`/llm/catalog`-Merge (`llmInfoApi.getModels`) + toter Modalitäts-Code werden gelöscht.

**Architecture:** Endpoint liefert zusätzlich `default` (für Vorauswahl). Die 6 Frontend-Aufrufer von `llmInfoApi.getModels()` migrieren auf `llmModelsApi.byModality("chat")`. Backend (validate, Buddy) ist seit SP1 schon unified. Danach: tote Alt-Pfade (MediaModelsSection, getModels, 4 Modalitäts-Helfer/Routen) entfernen. CatalogPage bleibt.

**Tech Stack:** FastAPI/pytest · React+TS+Vite (`npm run build` / `tsc -b`, NICHT `tsc --noEmit`).

**Spec:** `docs/superpowers/specs/2026-06-04-llm-ssot-v2-design.md`.

```python
@router.get("/models")
async def list_llm_models(
    modality: str | None = None,
    _: Annotated[tuple[str, str], Depends(require_auth)] = None,
) -> dict:
    """Kanonische Modell-Liste aus der Registry, optional nach Zweck gefiltert.
    Für ALLE Picker (require_auth, nicht admin-only). `default` = konfiguriertes
    Standard-Modell des Zwecks (für die Vorauswahl)."""
    entries = await registry.list_models(modality)
    if modality is None:
        purpose = "chat"
    elif modality in _config._PURPOSE_KEYS:
        purpose = modality
    else:
        purpose = None
    default = _config.get_default(purpose) if purpose else ""
    return {
        "default": default,
        "models": [
            {"id": e.id, "label": e.label, "provider": e.provider,
             "purposes": sorted(e.purposes), "context_window": e.context_window,
             "is_free": e.is_free, "embed_dim": e.embed_dim}
            for e in entries
        ],
    }
```
